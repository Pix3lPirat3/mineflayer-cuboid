### Getting Started

This plugin is built using Node and can be installed using:
```bash
npm install --save mineflayer-cuboid
```

### Require

```js
const Cuboid = require('mineflayer-cuboid');
var v = require('vec3');

let cube = Cuboid(v(0, 0, 0), v(5, 5, 5));

// Open src/index.js for functions
```
